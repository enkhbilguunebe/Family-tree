Нэвтрэх үг:
Лочин

Энэ хувилбарт зассан зүйл:
- Ургийн холбоосын логикийг шалгаж зассан.
- Эцэг/эхийн хосоос хүүхдүүд рүү гарах шугамыг хосын голоос гардаг болгосон.
- Шугамууд node дээр давхцахгүй, тойргийн ирмэг хүртэл тод хүрдэг болсон.
- Өлзийбүрэн + Гэрэл салааны бодит нэрс орсон.
- Цусан холбооны бүх гишүүний ургийн овог: Боржигон.
- Gerege мэдээллийн самбарт Овог, Хүүхэд, Хань хэсгийг хасаж Утасны дугаар нэмсэн.
- Утас дээр нэг хуруугаар чирж хөдөлгөх, хоёр хуруугаар zoom хийх logic хэвээр байна.

Одоогийн бүтэц:
1. Үүсгэн байгуулагч хос
2. Тэднээс 11 хүүхэд
3. 11 хүүхдийн нэг болох Өлзийбүрэн салаа үргэлжилнэ
4. Өлзийбүрэн + Гэрэл → 8 хүүхэд
5. 8 хүүхдийн дараагийн үе таны өгсөн нэрсээр орсон
